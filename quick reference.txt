NOTEBOOK SHORTCUTS
shift+enter - run cell + new
ctrl+enter - run cell
Esc, x - delete cell
up, down - select cell
ctrl+/ - (un)comment
----------------------------

[] => list
{} => dictionary
() => function
True/False => boolean

integer/float = number
function ~ worker
define ~ teach
argument/parameter = input
return ~ what to report back
attributes = features
index = address
